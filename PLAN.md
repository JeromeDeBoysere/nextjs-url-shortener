# Project 1: URL Shortener

## Description

Application to shorten long URLs into short links (like bit.ly) with click tracking.

---

## Features

- Form to create a short link
- Automatic unique slug generation (6 characters)
- Custom slugs (optional, if available)
- Automatic redirection to original URL
- Click counter
- Stats page with list of all links and click counts

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL (Supabase)
- Server Actions
- Middleware (redirections)
- Zod (validation)

---

## Learning Objectives

- Initial Next.js + database setup
- Unique identifier generation
- Dynamic redirections with middleware
- Basic CRUD without authentication
